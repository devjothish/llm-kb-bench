# Corpus

Karpathy's public material, used as the benchmark corpus.

## Contents

- **nanoGPT** - Training/finetuning medium-sized GPTs
- **micrograd** - Scalar-valued autograd engine
- **llm.c** - LLM training in raw C/CUDA
- **Blog posts** - Software 2.0, Training Recipes, Backprop, LeCun 1989

## Download

```bash
python -m corpus.download
```

Files are saved to `corpus/raw/`. This directory is gitignored.
